## Main Features: 
- [x] NextJS Typescript
- [x] TailwindCSS
- [x] Clerk Auth
- [x] Stripe Payment Gateway
- [x] Multi language using Zustand (indonesian & english)
- [x] Role based access control

### fill .env file

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

NEXT_PUBLIC_CLERK_SIGN_IN_URL=

UPLOADTHING_SECRET=

MUX_TOKEN_ID=
MUX_TOKEN_SECRET=
